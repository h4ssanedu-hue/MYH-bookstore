# Book Store Website

## Description
This is a simple Book Store Website where users can browse and view books online. The website is responsive and deployed live.

## Live Demo
```
https://h4ssanedu-hue.github.io/MYH-bookstore/
```

## Features
- Display list of books
- Category-wise browsing
- Book details page
- Responsive design
- Easy to use interface

## Technologies Used
- HTML
- CSS
- JavaScript
